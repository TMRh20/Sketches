# RF24 Advanced Interaction

Requires installed RF24 library and attached radio

### Description:

Advanced Example sketch to be used with RF24 GettingStarted example

1. Run the sketch, a browser window should pop up. If not, navigate to http://your-Q-IP-Address:7000

2. Use the Serial Monitor to select your radio number, 0 or 1

3. When data is sent to this device, it will display in the Serial Monitor, Python Console, and Web Browser

4. Use the button on the webpage to toggle the LED Matrix on/off
